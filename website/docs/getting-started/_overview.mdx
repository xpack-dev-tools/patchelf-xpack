The main target of this project is GNU/Linux, to adjust `DT_RPATH.

[NixOS PatchELF](https://github.com/NixOS/patchelf) is a simple utility
for modifying the dynamic linker and RPATH in existing ELF executables
and libraries.

**NixOS PatchELF** is an open source project hosted on
[GitHub](https://github.com/NixOS/patchelf).
