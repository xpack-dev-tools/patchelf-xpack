### Check the latest upstream release

The latest releases must have been announced in
NixOS PatchELF [GitHub Releases](https://github.com/NixOS/patchelf/releases/).

Compare the latest upstream release with the latest xPack
[release](https://github.com/xpack-dev-tools/patchelf-xpack/releases).
If necessary, update the triplet, otherwise increase the fourth number,
as below.
